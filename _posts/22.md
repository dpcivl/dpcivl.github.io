## 도커로 시작하는 분들을 위해 - 이거 보지 마세요;;
Yocto Project는 root 환경에서 빌드할 수 없다.  
그렇기 때문에 root 권한을 가진 다른 사용자 이름을 만들어서 docker 컨테이너를 만들려고 한다.  

텍스트 에디터를 열고 특정 디렉토리에 Dockerfile을 만든다.  
```Dockerfile
FROM ubuntu:20.04

# hu 사용자 생성 및 sudo 그룹에 추가
RUN useradd -m -s /bin/bash hu && \
    apt-get update && apt-get install -y sudo && \
    echo "hu ALL=(ALL) NOPASSWD: ALL" > /etc/sudoers.d/hu && \
    chmod 0440 /etc/sudoers.d/hu

USER hu
WORKDIR /home/hu

CMD ["/bin/bash"]
```  
🔼 나는 'hu'로 사용자 이름을 정했다.  

확장자 없이 'Dockerfile'로 이름 저장 후에,  
해당 디렉토리로 가서 아래의 명령어를 실행한다. 
```bash
docker build -t imx-ubuntu .
```  

위의 명렁어로 빌드가 완료되면 아래의 명령어를 실행해서 컨테이너를 만든다.  
```bash
docker run -it --name imx8mp imx-ubuntu bash
```
