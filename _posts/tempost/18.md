# 0) 개발 환경
- 리눅스 커널 : 'Linux raspberrypi 4.19.88-v7+ #1 SMP PREEMPT Wed Dec 11 14:34:49 KST 2019 armv7l GNU/Linux'
- 라즈베리파이CM3+

<br>

# 1) 첫번째 방법 실패
첫번째 방법은 `--user` 옵션을 사용하는 것이다.

```bash
pip install <패키지 이름> --user
```
🔼 위 명령어로 시도해보았지만 실패했다.  

# 2) sudo apt update && sudo apt upgrade
패키지가 최신 버전으로 업데이트 되지 않아서 그런가 해서 업데이틑 해봤다.  
안됐다.  

```bash
pip install --upgrade pip setuptools
```
🔼 위 명령어로 업그레이드를 해보려고 했지만 `pip install` 자체가 안되기 때문에 불가능했다. 
<br>

# 3) venv(가상 환경) 설치
```bash
python3 -m venv myenv
source myenv/bin/activate
```
🔼 위의 명령어 실행 이후 `pip install`을 하면 실행이 된다!!
