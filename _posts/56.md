# 1) 초기화
Network 클래스가 생성될 때 자동으로 호출된다.  
생성자로 전달된 `sizes` 리스트를 저장하고, 신경망의 층수를 변수에 저장한다.  

입력 층인 첫번째 층을 제외하고 각 층의 각 뉴런에 대한 bias 값을 무작위 값으로 초기화한다.  
레이어 사이의 연결 가중치를 저장하는 리스트 weights를 만든다.  
연속되는 두 레이어 사이의 뉴런 수를 쌍으로 묶어서 무작위 값으로 초기화한다.  

## 1> (y, x)
가중치 행렬의 형태는 일반적으로 '다음 층의 뉴런 수 x 현재 층의 뉴런 수'로 정의된다.  
이는 입력 데이터를 가중치 행렬과 곱할 때 ➡️ $wx + b$  
입력 데이터는 보통 열 벡터의 형태로 주어지기 때문에, 행렬 곱셈을 하기 위해서는 입력 데이터의 행 수와 가중치 행렬의 열 수가 일치해야 한다.  

<br>

# 2) feedforward
입력 데이터를 받아서 가중치 행렬 `w`와 행렬곱 연산을 수행한다.  
이후에 편향 벡터 `b`를 더하고 마지막으로 활성화 함수를 적용한다.  
해당 과정을 거쳐 나온 출력 데이터를 반환한다.  

<br>

# 3) SGD (Stochastic Gradient Descent)
training data의 순서를 섞은 후에, `mini_batches` 리스트에 훈련 데이터를 `mini_batches_size`만큼 나눠 할당한다.  
각 미니 배치가 끝나면 가중치와 편향을 업데이트 한다.  

## 1> 미니 배치를 이용한 가중치, 편향 업데이트
미니 배치에 있는 모든 훈련 데이터를 기반으로 하여, 신경망의 가중치와 편향 값이 업데이트된다.  

### 1] backpropagation
뉴런의 예측값인 `x`를 가지고 와서 가중치를 곱하고 편향을 더하여 `z`에 할당한다.  
`zs` 리스트에 `z`값을 넣고, `z`값에 활성화 함수를 적용한 값을 `activations` 리스트에 넣는다.
