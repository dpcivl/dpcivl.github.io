# 2)
pyqt5 패키지는 'qmake'라는 도구가 필요하다.  
이 도구가 시스템에 설치되어 있지 않다면 위의 오류가 발생하는 것 같다.  

```bash
qmake --version
```
🔼 'qmake' 있는지 확인
<br>

# 3) (필요 시) qmake 설치
'qmake'가 설치되어 있지 않다면 아래의 명령어를 실행해준다.  

```bash
sudo apt-get install qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools
```  


> 원래는 qt5-default를 install 해주면 됐는데 오류가 있다고 한다.
